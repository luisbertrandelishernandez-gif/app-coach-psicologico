Sección Hoy

La sección Hoy muestra el contenido del día según una rotación secuencial de 11 cuadernos CPVA. Cada día avanza al siguiente cuaderno en el ciclo.

  • Reflexión diaria: Texto del cuaderno del día. Aparece siempre. Puedes leerlo desplazándote por el visor de texto.
  • Podcast: Solo disponible lunes, miércoles y viernes (L/X/V). Usa el reproductor de audio integrado para escucharlo.
  • API key de Drive: Si el contenido no carga, introduce tu API key de Google Drive en el panel de configuración que aparece automáticamente. La key debe empezar por AIzaSy y tener 39 caracteres.

Aprendizaje CPVA

La sección Aprendizaje CPVA da acceso a los 9 módulos terapéuticos locales.

  • Elegir módulo: Pulsa cualquiera de las 9 tarjetas (ACT, Ansiedad, Apego, Crisis, Focusing, Ira, Logoterapia, Somática, TREC) para abrirlo.
  • Ejercicios: Pasos secuenciales numerados para practicar la técnica. Indicados para uso continuado.
  • Fichas rápidas: Resumen de consulta para recordar los conceptos clave de la técnica sin seguir pasos.
  • Badges de nivel:
    • Básico — ejercicios accesibles para cualquier persona.
    • Intermedio — requieren práctica previa.
    • Avanzado — para uso con supervisión profesional.
  • Etiquetas: Palabras clave descriptivas de cada ficha para identificar el contexto de uso.

Lectura en voz alta (TTS)

Pulsa el botón 🔊 junto a una reflexión, ejercicio o ficha para que la app lea el contenido en voz alta.

  • Detener: Vuelve a pulsar el botón 🔊 para parar la lectura en cualquier momento.
  • Cambiar velocidad: Usa el selector de velocidad de más abajo para ajustar el ritmo de lectura. El cambio se guarda automáticamente.
